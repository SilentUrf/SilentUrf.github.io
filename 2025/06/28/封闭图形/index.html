封闭图形

1.观察f1(x)=x^2+2x+1与f2(x)=-x+1的图象可知，f1与f2有两个确定的交点(0,1)和(-3,4)。注意到当-3≤x≤0时，f1与f2的图象构成一个封闭图形。
 
(1)求这个封闭图形的面积。
(2)试求证：在实数域定义两个连续的函数f和g（f≠g），若f,g有且仅有两个交点，则这f,g的图象在某一区间内可以构成一个封闭图形。
(3)试推广第(2)问的结论：在实数域定义两个连续的函数f和g（f≠g），若f,g有且仅有M个交点，则这f,g的图象在（M-1）区间内可以构成（M-1）个封闭图形，M为大于1的正整数。

(2)
证明：

考虑函数h(x) = f(x) - g(x)；由于f和g连续，h也连续。
令h(a) = 0，h(b) = 0，且f和g只有两个交点，因此在区间(a, b)内，h(x)≠0。

引理：连续函数的介值定理：闭区间上的连续函数可以取到最大值和最小值之间的任意值。

假设 h(x)在(a,b)内改变符号。那么，存在点 c,d ∈(a,b)，使得 h(c)和 h(d)异号。不失一般性，设 h(c)>0且h(d)<0(或反之，证明类似)。
由于h(x)在[a,b]上连续(作为闭区间上连续函数的限制)，且 c,d ∈(a,b)⊆[a,b]，由介值定理，在c和d之间(从而在(a, b)内)存在一点e，使得 h(e)= 0。
但这意味着 f(e)= g(e)，即(e,f(e))是f和g的另一个交点。由于e∈(a,b)，有e≠a且e≠b这与已知条件(f和g有且仅有两个交点)矛盾。因此，假设不成立，即 h(x)在(a,b)内不可能改变符号。
故 h(x)在(a,b)内不变号，即对于所有的x∈(a,b)，h(x) > 0或h(x) < 0。

不失一般性，假设在(a, b)内h(x) > 0，即f(x) > g(x)（反之，则g(x) > f(x)，类似证明）。
则在闭区间[a, b]上，有：

  • 在x = a和x = b处，f(x) = g(x)。
  • 在开区间(a, b)内，f(x) > g(x)。

因此，在区间[a, b]上，函数f和g的图像满足：在端点处相交，且在开区间(a, b)内，f的图像位于g的图像上方。这构成了一个封闭图形，具体为由曲线y = f(x)、y = g(x)及直线x = a、x = b围成的区域。

由于f和g连续，该区域是闭且有界的，故为封闭图形。因此，在区间[a, b]上，f和g的图像构成一个封闭图形。

综上，在实数域定义的两个连续函数f和g（f ≠g），若有且仅有两个交点，则存在某一区间，使得在该区间内其图像构成一个封闭图形。

(3)
证明：

假设 f 和 g 有且仅有 M 个交点，其中 M > 1 为正整数。记这些交点为 a1 < a2 < … < aM，满足 f(ai) = g(ai) 对于所有 i = 1, 2, …, M。

考虑相邻交点之间的闭区间 [ak, a{k+1}]，其中 k = 1, 2, …, M-1。在每个这样的区间上，分析函数行为：

  • 由于 f 和 g 连续，且仅在 ak 和 a{k+1} 处相等（因为无其他交点），在开区间 (ak, a{k+1}) 内，f(x)不等于g(x)。
  • 定义辅助函数 h(x) = f(x) - g(x)，其在 [ak, a{k+1}] 上连续。由介值定理，若 h(x) 在 (ak, a{k+1}) 内变号，则必存在零点，与无其他交点矛盾。因此，h(x) 在 (ak, a{k+1}) 内不变号，即在 [ak, a{k+1}] 上，要么 f(x) ≠g(x) 且等号仅在端点成立，要么 g(x) ≠f(x) 且等号仅在端点成立。
