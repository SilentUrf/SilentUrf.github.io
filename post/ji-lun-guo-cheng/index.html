及——论过程

2025-02-08 3 min read # 世论天下

 什么是永恒？
 永恒是期望，我们无时无刻盼望着永恒。时间是无情的长流，剥离了浮和沉。悬浮的一切无情地逐流远去，沉没的一切被无情地冲刷和掩盖。但是有情的人类注定不愿臣服于无情，于是盼望着名为永恒的力量作中流砥柱。
 什么能永恒？
 顽石般固执抵挡不了时过境迁的蹉跎，劲草般蓬勃抵挡不了地老天荒的零落。自然尚有海枯石烂之时，脆弱的生命谈何永恒？
 从未有停下亦或倒退，千千万万的孤帆在长流之中远去了，交错过之后又分离了。有朝一日，你的小船将超越了所有在你之前的帆船，你回头却发现与他们的距离如此遥远，犹如海中的一座又一座孤岛，在永远的停滞中逐渐化为一个亮点，你只能回头望见，却无法触及了。对于多少的人们，你又将是他们眼中的亮点呢？
 万物终有消逝之日，万物皆在消逝之中。
 而你能抓住什么呢？枯萎的花朵无法用双手的温度重现鲜活，垂老的人们无法用拥抱的力量重回少年。总是拼尽全力，才能在庆幸之中抓住稍纵即逝的绚烂。而后呢？火花熄灭，一切重新归于黑暗与平静，只留下分不清虚妄与现实的我们。待到消逝之后，抓住的也不过一把细沙，不必到风吹而流逝。
 难道希冀终作泡影，不过是自作多情，黄粱一梦？
 或许直到你触及褪色后的花瓣，抚摸焦黄的相片之后，才会明白——
 记忆便是永恒。
 一切是止不住的凋谢，可是你曾见过花朵绽放；一切是停不下的衰老，可是你曾见过青春歌唱。“有的人死了，他还活着”，血肉的消亡从来都不是终点，遗忘才是真正的消逝。记忆的相册又翻一页之际，你所紧握着的亦可放手，如蒲公英般自然的随风四散。当你真正超越一切，在船帆之下感到孤寂之时，回头望去，将会是漫天星辰，凝结在永恒的时刻。
&esmp;于是在漫长的旅途中，没有开始，没有结束，只有过程。
&esmp;在你所知的过程中，一切都是永恒；在过程外你所不知的地方，散落的你也将是过程中的永恒。

“离开世界之前，一切都是过程；
活着不难，最难的是做人；
在移开的眼神里，代表着默认；
这一切过程，我们曾经爱或恨。” ——《过程》蛋堡
